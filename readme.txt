<h1>채팅 매크로</h1>
<p>간편한 채팅 매크로 도구로, 반복적인 채팅 입력을 자동화합니다.</p>

<h2>사용 방법</h2>

<div class="step">
    <h3>1. <strong>main.exe 다운로드</strong></h3>
    <p>GitHub 저장소에서 <code>main.exe</code> 파일을 다운로드합니다.</p>
    
</div>

<div class="step">
    <h3>2. <strong>채팅에 입력할 내용 복사</strong></h3>
    <p>입력할 채팅 내용을 복사합니다. (Ctrl+C)</p>
    
</div>

<div class="step">
    <h3>3. <strong>실행 간격 설정</strong></h3>
    <p>실행 파일을 실행하고, 채팅 입력 간격을 초 단위로 설정합니다.</p>
    
</div>

<div class="step">
    <h3>4. <strong>프로그램 이동 후 시작</strong></h3>
    <p>매크로 시작 후, 자동 입력을 원하는 프로그램으로 이동합니다.</p>
    
</div>

<div class="note">
    <p><strong class="important">바이러스 경고:</strong> 일부 바이러스 감지 프로그램이 <code>main.exe</code> 파일을 잘못 감지할 수 있습니다. 파일이 안전한지 확인하고, 필요한 경우 예외 설정을 하십시오.</p>
